# Proyecto: Fundamentos de Manejo de Excepciones en Java

## 📌 Descripción

Este proyecto demuestra el uso básico del manejo de excepciones en Java mediante ejemplos prácticos.  
Se implementan distintos escenarios donde pueden ocurrir errores comunes en programación y se muestra cómo controlarlos correctamente para evitar que el programa se detenga de forma inesperada.

El programa principal (`com.example.App`) ejecuta todos los ejemplos implementados en las clases auxiliares, permitiendo visualizar en consola el comportamiento del manejo de excepciones.

---

## 🛠 Requisitos

- Java JDK 8 o superior
- (Opcional) Maven si el proyecto está estructurado como proyecto Maven
- IDE recomendado: VS Code, IntelliJ IDEA o Eclipse

## ▶️ Cómo Compilar y Ejecutar

1. Abrir el proyecto en el IDE.
2. Ejecutar la clase `App.java`.
3. Ver la salida en la consola.


## 📚 Tabla de Contenidos (Temas Implementados)

| Tema | Descripción |
|------|------------|
| Try-Catch | Manejo básico de excepciones usando bloques try-catch |
| División por cero | Control de ArithmeticException |
| Conversión inválida | Manejo de NumberFormatException |
| Bloque Finally | Uso del bloque finally para ejecución garantizada |
| Continuidad del programa | Prevención del cierre del programa tras una excepción |

---

## 🧠 Conceptos Demostrados

- Captura de excepciones específicas
- Uso correcto de bloques try, catch y finally
- Control del flujo del programa ante errores
- Manejo de errores sin detener la ejecución
- Organización del código en métodos estáticos

---

## ✅ Resultado Esperado

Al ejecutar el programa principal, se muestran ejemplos organizados por temas en la consola, demostrando que las excepciones son controladas correctamente y que el programa continúa ejecutándose después de cada error.
